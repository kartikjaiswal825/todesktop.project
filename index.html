FAQs

Is ToDesktop For Me?

That depends! If you would like to distribute your web app to your users as a downloadable desktop app then ToDesktop is for you.

What is the operating system?

Operating system acts as an interpreter between computer application and hardware. It works as a user interface.

What is IAS server?

AS means Internet Authentication Service. For many types of network access like wireless, authenticating switch and remote access dial-up.

What is a Ping command?

Between two or more devices, to check the physical IP connectivity Ping command is used..

What is OU?

OU means for Organizational Unit. It is a container within Active Directory which can hold users, groups, and computers.
